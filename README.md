# JobFinder

A modern job search application built with Angular 17+ and Tailwind CSS.

## Features
- **Job Search**: Integrated with The Muse API.
- **Favorites**: Save jobs to your favorites list (NgRx).
- **Application Tracking**: Track your job applications.
- **Authentication**: User login and registration.
- **Profile**: Manage your user profile.

## Getting Started

### Prerequisites
- Node.js (v18+)
- Angular CLI

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application
1. Start the backend (JSON Server):
   ```bash
   npm run server
   ```
2. Start the frontend:
   ```bash
   ng serve
   ```
3. Navigate to `http://localhost:4200/`

## Architecture
- **Frontend**: Angular 17 (Standalone Components)
- **State Management**: NgRx
- **Styling**: Tailwind CSS
- **Backend**: JSON Server (Mock API)
